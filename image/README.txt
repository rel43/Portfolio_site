How to add images
=================

Put photos or drawings in this folder and rename them exactly like the file
names below. If the name is different by even one letter, the site will not
find it.

Example:
If you want to change the round portrait on the main page, put the photo here
and rename it to:
profile.jpg

Main page / odin.html
---------------------
profile.jpg
  Round portrait in the left menu. Best: square photo, face in the center.

Important notes
---------------
- The easiest format is JPG for photos and PNG for QR codes or transparent
  drawings.
- Keep the file names exactly as written here.
- Do not put spaces or Russian letters in these file names.
- If a file is missing, the site will show the old internet image or a soft
  placeholder.
- If you replace an image and the browser still shows the old one, open the
  page with ?fresh at the end:
  http://127.0.0.1:8000/odin.html?fresh
